ChiDogs

Capstone Project

I am currently working through CORS deployment bugs with Heroku. Unfortunately, the site will not function properly at the moment, but feel free to check out the repos.


As a loving dog dad of an incredibly social boxer, dog parks are a regular destination for Duckie and I. But, all too often we would go to a park and there were very few or sometimes no other dogs there at all. This is where ChiDogs comes in. A public message board for dog owners to plan and post about the dog parks that they are going to be visiting the day of or on future dates!

This web application was created using Python, Django, PostgreSQL, and React. It allows the user to create a profile, add their dogs to their profile, add photos of their dogs, create posts about where they are planning to go and view the posts of other users. It also allows the user to view other user’s pets to see if they are compatible with their own dog. I hadn’t really worked much in React and controlling state was very tricky, but I definitely enjoyed the challenge. I would like to go back and create a global state and get away from using React Router. I also want to integrate Google Maps API and allow users to add events to their calendars.

Budgeter

Personal Project

This project is going to be a full stack application that allows users to create and manipulate custom budgets. The user will create a secured account which they can save budgets and create scenarios using those created budgets. Want to know what would happen if you saved an extra $50 a paycheck for a year? How would it impact your monthly available cash if you spent less on going out? All of this information will be displayed using data visualization tools like Chart.js!
